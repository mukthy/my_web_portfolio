002 : SlackApp with Flask

"It is a small Flask based web-app that connects internal tools API of the company to Slack."

My company wanted to access some of the internal tools from Slack itself using a slack-command. All these internal tools had built-in API. The only part that I had to do was capture the SlackCommand with the action and forward that action to the internal API tools and give the result back on the slack itself.

So I created a small Flask based web-app which acts as a bridge between the internal tools API and Slack, when ever someone enters a /slack-command in company's slack-channel it will be sent as a POST request to my Web-App and that performs the action such as sending it to the Internal API and get the results then posts it back to Slack as a result.

This helps users to simply access the tools from their phone itself as they have slack on phone and all they need is to send /slackcommand on slack-channel
